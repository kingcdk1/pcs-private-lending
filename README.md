# PCS Private Lending — Landing Page

Real estate loan application landing page for PCS Tax Service and Academy for Business.

## Setup

1. Push this repo to GitHub
2. Import into Vercel
3. Replace `YOUR_GHL_WEBHOOK_URL` in `public/index.html` with your GoHighLevel inbound webhook URL
4. Redeploy

## Features

- 3-step loan application form
- Meta Pixel tracking (both PCS pixel IDs)
- GoHighLevel webhook integration
- Mobile responsive
- Auto phone/currency formatting
- Form validation
